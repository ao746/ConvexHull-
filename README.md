# Convex Hull

First university project based on a Computational Geometry practice focused on Convex Hull algorithms.

Beyond implementing the algorithms themselves, the main goals of this project were:
- First hands-on experience building a **graphical user interface** in Java
- First approach to building a **runnable Java application** (.jar)

## Algorithms implemented
- Brute Force
- Jarvis March
- QuickHull
- Graham Scan
- MergeHull


*Some parts of this project were developed with the assistance of AI tools, used as a learning and support resource throughout the process.*
